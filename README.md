# e-commerce
API para um e-commerce desenvolvido durante o Curso da Cubos Academy

## Endpoints

### POST - /cadastro :

#### Dados enviados

- Email
- Senha
- Nome da loja

#### Dados retornados
- Sucesso / Erro

### POST - /login 

#### Dados enviados

- senha
